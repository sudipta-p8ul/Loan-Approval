# Loan-Approval
# 🏦 Loan Approval Prediction

This project focuses on predicting whether a loan application will be approved or not using Machine Learning models. It helps financial institutions automate decision-making and reduce risk by analyzing applicant data.

---

## 📊 Models Used

### 1. Logistic Regression
- Accuracy: **0.9133**
- Sensitivity (Recall): **0.9416**
- Specificity: **0.8669**
- F1 Score: **0.9311**
- AUC: **0.9726**

### 2. Support Vector Machine (SVM)
- Accuracy: **0.9450**
- Sensitivity (Recall): **0.9529**
- Specificity: **0.9319**
- F1 Score: **0.9556**
- AUC: **0.9865**

---

## 📈 Key Insights

- **SVM outperforms Logistic Regression** in all evaluation metrics.
- High **AUC scores (>0.97)** indicate excellent model discrimination.
- Strong **F1 Scores** show a good balance between precision and recall.
- The model is highly effective for real-world loan approval systems.
